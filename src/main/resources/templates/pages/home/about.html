O aplikaci

Nacházíte se ve fiktivní aplikaci nabízející správu pojištěných osob a jejich pojištění. Jde o projekt, který sloužil k naučení základních principů Spring Boot a MVC architektury

Nabízí pomocí Spring security možnost přihlášení přes dvě uživatelské entity: "klient" a "zaměstnanec", s nastavenými odlišnými právy pro správu uložených záznamů.

Ke každé entitě umožněny základní CRUD operace (vytvoření, zobrazení, editace, mazání)

K zobrazení v HTML použit šablonovací systém Thymeleaf, ke stylování použit framework Bootstrap
